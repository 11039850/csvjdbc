CsvJdbc - a JDBC driver for CSV files
Release notes

Version 0.7
Release date: 2001-12-2

Notes: Added possibility to pass properties to driver.

Changes: 

  • Use Drivermanager.getConnection(String url,Properties info) to pass properties to driver.
  • Property seperator added to specify a different seperator (Default is ',').
  • Property suppressHeaders added to specify if the first line contains header information (Default is false; headers are on first line, do not suppress).
  • Property extension added to specify a different extension (Default is ".csv").
